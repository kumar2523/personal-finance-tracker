# 💰 Personal Finance Tracker

A responsive web application to manage income and expenses, track balance, and visualize spending using charts.

---

## 🚀 Live Demo

👉 https://adorable-manatee-f0cc06.netlify.app

---

## 📌 Features

* ➕ Add income and expense transactions
* ✏️ Edit existing transactions
* ❌ Delete transactions
* 🔍 Filter transactions (All / Income / Expense)
* 🧹 Clear all transactions
* 💾 Persistent data using LocalStorage
* 📊 Expense visualization using Pie Chart (Chart.js)
* 📁 Export transaction data as CSV
* ✅ Input validation for accurate data entry

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* LocalStorage
* Chart.js
* Netlify (Deployment)

---

## 📊 How it Works

1. User adds a transaction (income or expense)
2. Data is stored as objects in an array
3. Array is saved in LocalStorage
4. UI updates dynamically
5. Summary (income, expense, balance) recalculates
6. Chart updates automatically based on expenses
